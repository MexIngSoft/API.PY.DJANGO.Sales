# API.PY.DJANGO.Sales

API Django responsable de cotizaciones, pedidos y flujo comercial.

## Estandar local

- Nomenclatura actual: `API.PY.DJANGO.Sales`.
- Nomenclatura nueva para proyectos futuros: `API.PY.DJANGO.Sales`.
- Paquete de settings actual: `config`.
- App principal: `sales`.
- Archivo de entorno local: `.env.local`.
- Plantilla segura: `.env.local.example`.
- Dependencias: `requirements.txt`.

## Comandos basicos

```sh
python manage.py migrate
python manage.py runserver
```
